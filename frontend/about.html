About Nykaa Demo Shop

Your one‑stop online fashion & lifestyle destination — created as a demo project for learning full‑stack web development with clean UI and smooth user experience.

What We Sell

Trendy Western wear, classy Indian traditional outfits, kids fashion, party looks, wedding categories and much more — thoughtfully arranged for easy browsing.

Why This Project

This project is built to practice HTML, CSS, JavaScript, responsive layouts, product pages, navigation flow, and UI/UX concepts — exactly like a real e‑commerce site.

Our Goal

To create a modern shopping experience with smooth scrolling, category pages, search feature, login, and cart flow — giving a complete beginner‑friendly portfolio project.

Our Story

Nykaa Demo Shop started as a small practice assignment and soon grew into a complete mini e‑commerce experience. The idea was simple — make a clean, attractive website that helps beginners understand the structure of a real shopping platform.

From category pages and banners to search bar, scroll sections, contact page, and cart setup — each part is crafted with learning and creativity in mind. This project continues to evolve with new pages, improved UI, and modern features.
